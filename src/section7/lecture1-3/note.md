# 몽고디비 설치하기

## 몽고디비

### NoSQL

SQL데이터 베이스와는 다른 유형의 데이터

SQL(MySQL) | NoSQL(몽고디비) |
---------|----------|
 규칙에 맞는 데이터 입력 | 자유로운 데이터 입력 |
 테이블 간 JOIN 지원 | 컬렉션 간 JOIN 미지원 |
 안정성, 일관성 | 확장성, 가용성 |
 용어(테이블, 로우, 컬럼)       |   용어(컬렉션, 다큐먼트, 필드)

- JOIN : 관계가 있는 테이블끼리 데이터를 합치는 기능
- 빅데이터, 메시징, 세션 관리 등(비정형 데이터)에는 몽고디비 사용성이 좋음
