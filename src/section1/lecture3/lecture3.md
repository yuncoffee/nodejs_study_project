# var, const, let

- var
-------------------- 스코프 차이  var 블록 무시. const 는 불록 밖에서 못 씀.
-------------------- 단, function 안에선 무시 못함.
- const
-------------------- const 에 값 선언할 땐 맨처음 한번만 가능     const a = 3 ;  a = 4 (X)
-------------------- 객체를 넣었을 땐 변경은 가능.   
-------------------- let 은 됨.
- let 


const로 선언을 해야 값 바뀔일이 적어짐. 필요시 let
