# 예외 처리하기

## 노드 내장 모듈 - 10

### 예외처리

exception, error?

js에서는 큰 차이 없음

- 노드 스레드가 멈춤
- 노드는 기본적으로 싱글 스ㅔㄹ드가 스레드가 멈춘다 -> 프로세스가 멈춘다

기본적으로 에러 발생할 곳에 try catch로 예외를 처리하라

**언제 감쌈?**

- 콜백 함수에서 에러 객체 제공 시 안해도 됨 (로그는 찍어주자)
- promise에서 안 해도 안 멈춤(catch 붙여라 제발)

**프로세스 객체에서 모든 에러 모으기**

- 콜백 함수의 동작 보장 x
- 복구 작업으로는 부적합
- 에러 내용 기록 용으로 활용이 좋음

**에러가 발생햇다?**

- 최대한 빨리 에러 발생을 깨닫는다.
- 프로세스를 종료한다.
- 에러를 (매우 빠르게) 처리한다.
- 서비스를 재시작한다.
